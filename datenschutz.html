2. Welche Daten werden verarbeitet?

2.1 Lokale Daten (nur auf deinem Gerät)

Folgende Daten werden ausschließlich im lokalen Browserspeicher (localStorage / IndexedDB) deines Geräts gespeichert und sind für uns nicht zugänglich:

  • Lizenznummer – zur Validierung der Berechtigung
  • Gemini API Key – dein persönlicher Google-API-Schlüssel
  • Kategorien und Konfiguration – deine persönliche Einrichtung
  • Warteschlange und Protokoll – Dateinamen verarbeiteter Dateien (keine Inhalte)
  • Ordnerhandle – Referenz auf deinen gewählten Arbeitsordner (IndexedDB)

2.2 Daten die Google Gemini erhält

Für die KI-Analyse wird der Inhalt deiner Dokumente an die Google Gemini API gesendet. Dies geschieht direkt von deinem Browser an Google – nicht über unsere Server. Google verarbeitet diese Daten gemäß eigenen Datenschutzbestimmungen.

Empfehlung: Keine hochsensiblen Dokumente (Ausweise, Passwörter, medizinische Akten) ohne Kenntnis von Googles Datenschutzpraktiken verarbeiten. Mehr unter policies.google.com/privacy.

2.3 Lizenzvalidierung (Supabase)

Beim App-Login wird deine Lizenznummer zur Validierung an unsere Supabase-Backend-Infrastruktur gesendet. Es werden dabei keine persönlichen Daten übermittelt – nur der Lizenzschlüssel selbst. Supabase ist ein EU-konformer Dienst.

2.4 Hosting (GitHub Pages)

Die App wird über GitHub Pages bereitgestellt. GitHub kann technische Zugriffsdaten (IP-Adresse, Browsertyp, Zeitstempel) protokollieren. Siehe GitHub Datenschutzerklärung.

2.5 Beta-Bewerbung

Bei einer Beta-Bewerbung werden Vorname, Nachname, E-Mail-Adresse und optionale Angaben (Betriebssystem, Dokumentenvolumen, Nachricht) in unserer Supabase-Datenbank gespeichert. Diese Daten dienen ausschließlich der Verwaltung des Beta-Programms und werden nicht an Dritte weitergegeben. Ein gehashter IP-Adress-Wert wird zur Spam-Prävention gespeichert (kein Klartext).

  • Art. 6 Abs. 1 lit. f DSGVO – berechtigtes Interesse (technische Bereitstellung, Spam-Schutz)

4. Datenweitergabe

Keine Weitergabe an Dritte außer:

  • Google LLC – Dokumenteninhalte für KI-Analyse (Gemini API)
  • GitHub Inc. – technische Zugriffsdaten durch Hosting (GitHub Pages)
  • Supabase Inc. – Lizenzvalidierung und Beta-Bewerbungsdaten

5. Speicherdauer

Lokale Browserdaten bleiben gespeichert bis du sie aktiv löschst (Einstellungen → Lizenz entfernen → alle Daten löschen). Beta-Bewerbungs-Daten werden nach Abschluss des Beta-Programms oder auf Anfrage gelöscht. Lizenz-Daten werden für die Dauer der Lizenzlaufzeit plus gesetzlicher Aufbewahrungsfristen gespeichert.

7. Cookies und Tracking

DokumentenOrganizer setzt keine Cookies und verwendet kein Tracking, keine Analyse-Tools und keine Werbenetzwerke.

8. Datensicherheit

Die Kommunikation zwischen App und externen Diensten (Gemini API, Supabase) erfolgt ausschließlich über verschlüsselte HTTPS-Verbindungen. Deine Dokumente verlassen niemals dein Gerät in Richtung unserer Server.
